  • Fixed an error where if a queue with non existing id was opened, an unexpected exception was thrown.

2.1.0

  • Thresholding. It is now possible to set a time duration and a maximum amount of jobs that can be executed by a queue. The queuing system will make sure that the jobs will be executed balanced over that timespan.
  • Chaining. Methods on objects can now be chained.
  • Implicit queue creation. Queues will now implicitly be created when an action is performed on a non-existent queue. It is possible to turn this feature of from the config file
  • Loading from constructor. Loading a queue or a job can now be done from the constructor.
  • PHPDoc documentation is added
  • Composer.json is updated and made better.
